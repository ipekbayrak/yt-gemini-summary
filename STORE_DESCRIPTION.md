# Chrome Web Store Description

## Short description
One-click YouTube video summaries in Gemini. Adds a button to cards, fills the prompt, optional auto-send.

## Detailed description
YouTube to Gemini Summary adds a "Summarize with Gemini" button to YouTube video and Shorts cards.
Click it to open Gemini, fill a ready-to-send prompt, and optionally auto-send.

Features:
- Works on Home, Search, Subscriptions, Watch sidebar, and Shorts.
- Customizable prompt template with {title}, {channel}, {url}.
- Options for language, auto-send, open in new tab, hover-only, and send delay.
- No external servers. Uses only local browser storage.

Permissions:
- tabs: open or focus the Gemini tab.
- storage: save options and pending prompt.
- host permissions: run on YouTube and Gemini.
